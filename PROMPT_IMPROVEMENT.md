# User Insight プロンプト改善: 直近3件重視の明確化

## 改善内容

ユーザーが「直近3件を重視してみてくれている」ことを実感できるよう、プロンプトを大幅に強化しました。

---

## 🎯 主な変更点

### 1. **視覚的な強調**

**Before:**
```
### 【最重要】直近3件のコーディネート
```

**After:**
```
### 🔥【最重要】直近3件のコーディネート（ユーザーの現在のスタイル）
**この3件から現在のスタイル傾向を必ず分析してください！**
```

- 絵文字（🎯🔥📅📚）で視覚的に強調
- 「現在のスタイル」と明記して重要性を強調
- 「必ず分析してください」と明示的に指示

---

### 2. **分析の優先順位を明記**

**追加セクション:**
```
**🎯 分析の優先順位:**
1. **直近3件を最重要視** - ユーザーの現在のスタイル傾向を表す
2. 4-7件目は参考情報 - 過去のスタイル傾向との比較に使用
```

- Geminiに優先順位を明確に伝達
- 直近3件と4-7件目の役割を明確化

---

### 3. **最重要ルールの追加**

**Before:**
```
**出力ガイドライン:**
- ファッションタイプ、動物占い、そして実際のコーディネート履歴（特に直近3件を重視）の全ての情報を統合
```

**After:**
```
**🎯 最重要ルール:**
1. **必ず「直近3件のコーディネート」を中心に分析してください**
2. 直近3件から見える具体的な傾向（色、テイスト、アイテム、タグ）を必ず言及
3. 「最近のあなたは〜」「直近3件のコーデを見ると〜」など、時系列を意識した表現を使用
4. 可能であれば、過去のコーデ（4-7件目）との比較・変化も言及
```

- 直近3件を中心に分析することを最重要ルールとして明記
- 具体的な要素（色、テイスト、アイテム、タグ）の言及を義務化
- 時系列表現の使用を指示

---

### 4. **時系列表現の義務化**

**出力ガイドラインに追加:**
```
- **冒頭で「直近3件のコーデ」の分析から始める**（ユーザーに実感させる）
- 直近3件に共通する要素を具体的に抽出（例：「直近3件すべてにモノトーンが登場」）
- 直近3件の具体的なタグやキャッチフレーズを引用
```

- インサイトの冒頭で直近3件に言及することを義務化
- 共通要素の抽出を指示
- 実際のタグ・キャッチフレーズを引用するよう指示

## 💡 ユーザーが実感できる理由

### 1. **冒頭で明示**
インサイトの最初の一文が必ず「直近3件のコーデ」に言及するため、ユーザーは最初から「ちゃんと最近のコーデを見てくれている」と感じる。

### 2. **実際のデータを引用**
実際のキャッチフレーズやタグを引用することで、「本当に自分のコーデを見ている」という実感が得られる。

### 3. **時系列表現**
「最近のあなたは〜」「直近3件を見ると〜」といった表現により、「現在」のスタイルを分析していることが明確。

### 4. **共通要素の抽出**
「直近3件すべてに黒が登場」「3件中2件にレイヤード」など、具体的な共通点を示すことで、深い分析をしていることが伝わる。

### 5. **過去との比較**
4-7件目のデータと比較して変化を言及することで、時系列での変化を追っていることが実感できる。
